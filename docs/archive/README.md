# Archive

This folder is the durable home for historical material that no longer belongs
at repo root or in active layer/service canon.

Top-level contract:
- keep only dated rollups named `YYYY-MM-<slug>.md`
- keep `README.md` as the archive entrypoint
- do not leave one-file-per-packet copies or `.old` files at top level

Migration rule:
- temporary leftovers belong under `docs/archive/legacy/`
- new archive work should be rollups, not raw packet dumps

Current truth still lives in active canon:
- `docs/PLATFORM_DOSSIER.md`
- `docs/INTEGRATIONS.md`
- `docs/foundation/`
- layer/service `SERVICE_SPEC.md`, `RUNBOOK.md`, and `README.md`
